Resumo

Acreditamos que acesso a informações de emergência é um direito básico, não um produto. Por isso, o GlobalSOS funciona completamente sem que você precise criar conta, fazer login ou fornecer qualquer dado pessoal.

Este documento detalha exatamente o que acontece — e o que não acontece — com suas informações quando você usa nosso site ou aplicativo.

O que coletamos

Para que o site e o app funcionem corretamente, os seguintes dados técnicos são processados localmente no seu dispositivo e nunca enviados a servidores externos:

  • Idioma do sistema — detectado pelo navegador para exibir o site no idioma correto automaticamente. Não é armazenado em nossos servidores.
  • Preferência de idioma — se você troca o idioma manualmente, essa preferência é salva apenas no seu dispositivo via localStorage.
  • IP para detecção de país — ao carregar o site, fazemos uma consulta anônima ao serviço ipapi.co para sugerir o idioma baseado no seu país. Nenhum dado seu é armazenado por nós nesse processo.

O que não coletamos

A lista do que nunca coletamos é mais importante do que o que coletamos:

🚫
Nenhum dado cadastral

Nome, e-mail, CPF, telefone — não pedimos, não coletamos, não armazenamos.

🚫
Nenhum histórico de buscas

O que você pesquisa permanece no seu dispositivo. Nós não sabemos o que você buscou.

🚫
Nenhuma localização persistida

O GPS é usado só para identificar seu país. Sua localização não é enviada nem armazenada.

🚫
Sem cookies de rastreamento

Não utilizamos cookies de terceiros, pixels de rastreamento ou ferramentas de retargeting.

🚫
Sem perfis de usuário

Não existe um "perfil seu" nos nossos sistemas. Você é completamente anônimo para nós.

🚫
Sem venda de dados

Não monetizamos dados de usuários. Nunca. Nosso modelo não depende disso.

Sobre o Site Web

O GlobalSOS Web é construído em HTML, CSS e JavaScript puro, sem dependências de rastreamento. Especificamente:

  • Sem Google Analytics ou qualquer ferramenta similar de analytics.
  • Sem Meta Pixel, Google Ads ou qualquer pixel de terceiros.
  • Sem cookies de terceiros. O único dado armazenado localmente é sua preferência de idioma (gsos_lang no localStorage).
  • Detecção de idioma por IP: ao carregar a página, uma requisição anônima é feita ao ipapi.co para identificar seu país e sugerir o idioma correto. Essa consulta é feita pelo seu navegador diretamente ao serviço, sem intermediação nossa. Consulte a política de privacidade do ipapi.co.
  • Fontes externas: as fontes tipográficas são carregadas pelo Google Fonts. Veja a política de privacidade do Google.

Base de Dados de Números

Os números de emergência exibidos no GlobalSOS são informações de domínio público, compiladas de fontes oficiais dos governos e serviços de emergência de cada país.

A base de dados é revisada e atualizada periodicamente para garantir precisão. Se você identificar um número incorreto, entre em contato com a equipe.

Seus Direitos

Como o GlobalSOS não coleta dados pessoais identificáveis, não existem dados pessoais seus armazenados em nossos sistemas para acessar, corrigir ou excluir.

Caso você resida em uma jurisdição com legislação de proteção de dados (como a LGPD no Brasil, o GDPR na União Europeia ou a CCPA na Califórnia), saiba que o GlobalSOS foi projetado para estar em conformidade com esses regulamentos por padrão — simplesmente por não coletar os dados que eles protegem.
